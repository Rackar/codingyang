# 航空影像快速质检

# 介绍

仅使用概略 POS，快速评定航摄飞行质量。航摄质量检查的主要方法为传统人工检查法和软件自动检查法。人工检查耗时较长、效率偏低、容易漏检。自动检查效率较高，但需要准备相应的 POS 数据、影像数据和其他参数文件，其中影像数据需要一定时长，进行拼接匀色等处理才能得到结果。本软件仅使用 POS 数据直接定向，来快速评估飞行质量，定位可疑点。
